# Izaija

Izaija is a web app for mock investments. You can make real-time investment picks with virtual money, track your positions, and later see how your decisions performed.

The goal is to make investing practice simple, low-risk, and measurable without using real funds.

## Planned Features

- Real-time mock investment tracking
- Virtual cash balance and portfolio history
- Performance summaries for past decisions
- Watchlists for assets and ideas
- Simple dashboards for gains, losses, and open positions
